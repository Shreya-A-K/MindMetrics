# 🧠 MindMetrics: Attention Analysis System

**MindMetrics** is an interactive web-based project that analyzes user attention span through behavioral data collected from a *Simon Says* game.

The system tracks real-time user interactions such as reaction time, accuracy, and progression level, and transforms this raw data into meaningful insights using machine learning techniques.

---

## 🚀 Features

* 🎮 Interactive Simon Says game
* ⏱️ Reaction time tracking
* 🎯 Accuracy and performance analysis
* 📊 Behavioral data collection via event logging
* 🧠 Attention score computation
* 🤖 Machine learning for pattern detection (clustering / prediction)
* 📈 Visual analytics for attention trends

---

## 🧩 How It Works

1. The user plays a Simon Says game in the browser
2. Event listeners capture interaction data (clicks, timing, correctness)
3. Data is sent to a Python (Flask) backend and stored
4. Raw logs are processed into structured features
5. Machine learning models analyze attention patterns
6. Results are visualized to provide insights into user focus and behavior

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Data Handling:** JSON / Pandas
* **Machine Learning:** Scikit-learn
* **Visualization:** Chart.js / Matplotlib
